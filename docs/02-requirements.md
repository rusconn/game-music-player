# 要求・要件

## 要求

- ゲーム音楽oggのループ再生
- 非ループoggも再生
- その他主要なフォーマットも再生
- 複数ファイルの一括登録
- シーク
- 音量調整
- 再生速度・音程の変更
  - たまにゲーム側で再生速度を変更していることがある
- 各種設定の自動保存
  - いちいち設定し直さなくていい
- イヤホンの再生・一時停止ボタンからのコントロール
  - プレイヤーの画面を探し出して操作しなくていい
- スマートフォン対応
  - 出先でも使えるといい
  - ネイティブっぽい体験
- シンプルなUI
  - 多機能は求めていないので簡単に使えることを優先
  - アートワーク不要
  - グルーピング不要

## 要件

- PWAでPCとスマートフォンに対応
  - PWAのインストールでネイティブのようなUIを提供
- シングルカラムUI
- `<input type="file" multiple>`で一括登録
- 各種再生、oggループ、シーク、音量・再生速度変更等はWeb Audio APIで実現
- WebStorageへ設定を保存して利用する
- MediaSession APIでOSのメディアコントロールと連携する
