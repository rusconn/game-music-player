# オーディオ実装メモ

- Web Audio APIを採用
  - audio要素ではスムーズなoggループを実現できなかった
  - 代償として、audio要素の提供する各種機能を自前実装することとなった
- OSのメディアコントロールを利用するため、MediaSession APIを採用
  - ただしWeb Audio APIによるオーディオ再生ではメディアコントロールが反応しなかった
  - ワークアラウンドとしてaudio要素に無音ファイルを再生させ、Web Audio APIによるオーディオ再生と連動させた
  - もっと良いやり方があるのではないか…？
- loopEndよりも後ろへシークした際の挙動
  - 一旦曲全体をループし、その後範囲ループへ戻る仕様とした
  - この仕様のおかげで実装が相当に複雑化した
